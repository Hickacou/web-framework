Welcome on your Web Framework !

To understand how it works, read this little Guide !

Framework Settings

In index.html you can find the Framework's settings :

  • FRAMEWORK_CLASS : Represents the class name of the sections to be recognized by the Framework and put in the left menu. By default : FW_SECTION.
  • SHOW_SECTION_NAME : Sets the display of the section name in the corner. If True, sections must have a display-name attribute. True by default.
  • CSS_PATH : The path to the directory containing your CSS files
  • JS_PATH : The path to the directory containing your JS files

Framework Sections attributes

Framework's sections have several attributes :

  • display-name : The name to display in the menu. If empty, id is used instead.
  • script : The name of the optionnal script to use for this section.
  • css-name : The name of the CSS file linked to this section. If empty, id is used instead.
  • center : If true, centers all the content inside of the section. False by default.
